# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-07-09

### Added

- Added integration brand assets (`icon.png` and `logo.png`).
- Minor documentation improvements.

## [1.0.0] - 2026-07-09

### Added

- Initial public release.
- Support for FANN EkoTreat.
- Support for FANN Biobed Control Unit.
- Config Flow for setup through the Home Assistant UI.
- Wake/Sleep control.
- Status sensor.
- Connection binary sensor.
- EkoTreat People sensor.
- Biobed Schedule sensor.
- Refresh button.
- Automatic session handling and re-authentication.
- Diagnostics support.
